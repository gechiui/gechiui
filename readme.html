安装: 快速安装引导

  1. 将包解压缩到一个空目录中并上传所有内容。
  2. 在浏览器中打开 gc-admin/install.php 。它将带您完成设置 gc-config.php 包含数据库连接详细信息的文件的过程。
    1. 如果由于某种原因这不起作用，请不要担心。它不适用于所有网络主机。 gc-config-sample.php 使用 WordPad 或类似的文本编辑器打开并填写您的数据库连接详细信息。
    2. 将文件另存为 gc-config.php 并上传。
    3. 在浏览器中打开 gc-admin/install.php 。
  3. 设置配置文件后，安装程序将为您的站点设置所需的表。如果有错误，请仔细检查您的 gc-config.php 文件，然后重试。如果再次失败，请访问 GeChiUI 支持 并收集尽可能多的数据。
  4. 如果您没有输入密码，请记下提供给您的密码。 如果您没有提供用户名，它将是 admin。
  5. 然后安装程序应该将您发送到 登录页面。 使用您在安装过程中选择的用户名和密码登录。如果为您生成了密码，您可以单击“个人资料”更改密码。

更新

使用自动更新程序

  1. 在浏览器中打开 gc-admin/update-core.php并按照说明进行操作。
  2. 也许你想要更多？这就够了！

手动更新

  1. 在更新任何内容之前，请确保您拥有可能已修改的任何文件的备份副本，例如 index.php.
  2. 删除旧的 GeChiUI 文件，保存已修改的文件。
  3. 上传新文件。
  4. 将浏览器指向 /gc-admin/upgrade.php.

从其他系统迁移

GeChiUI 可以 从多个系统导入。首先，在使用 导入工具之前，您需要安装 GeChiUI 并按上述方式工作。

系统要求

  • PHP version 5.6.20 或更高版本。
  • MySQL version 5.0 或更高版本。

建议

  • PHP version 7.4 或更高版本。
  • MySQL version 5.6 或更高版本。
  • 使用 mod_rewrite Apache模块。
  • 使用 HTTPS
  • 保留您网站上的 www.gechiui.com 链接。
